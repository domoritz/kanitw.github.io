Becoming a Sublime (Text) Ninja

With Sublime Text 2, I can forget about trying vim, textmate. To utilize the full power of sublime, we need to setup some stuffs and install extensions.

Setting up for command line

If you haven’t add ~/bin to your path, then add.

echo 'export PATH=~/bin:$PATH' >> ~/.profile

Then

defaults write ~/.MacOSX/environment EDITOR "~/bin/subl -w "

Shortkeys

The default short keys that comes with Sublime Text 2 are great. But there are some useful addition that I think it is helpful as well.

[
 { "keys": ["cmd+alt+r"], "command": "reindent" , "args": {"single_line": false}}
 ,
 { "keys": ["cmd+shift+k"], "command": "run_macro_file", "args": {"file": "Packages/Default/Delete Line.sublime-macro"} }
 ,
 { "keys": ["super+alt+f"], "command": "show_panel", "args": {"panel": "find"} }
 ,
 { "keys": ["super+f"], "command": "show_panel", "args": {"panel": "replace"} }
 ,
 { "keys": ["ctrl+shift+t"], "command": "delete_trailing_spaces" }
]

Plugins

Install Package Control  first. Then you can easily install any of the following good plugins:

  • General Purpose

    • SublimeCodeIntel – super cool autocompletion

    • AllAutoComplete – extend sublime autocomplete to check all opened files.

    • Alignment

    • BracketHighlighter

    • Sidebar Enhancement - many cool add-ons to the sidebar.

    • EasyMotion super fast navigation with cmd+;+<character-to-jump-to>–a super plugin inspired from Vim

    • sublimelinter – Great for checking syntax errors and warning for most popular languages

    • ChangeList Go to next edited/last edited location.

    • SublimeTODO - allow you to generate TODO list from TODO comments

    • docblockr – creating documentation block really easy!  (/** enter = done!)

    • AdvancedNewFile - create new file anywhere with cmd+opt+n

    • Nettuts Fetch – Quickly Fetch files here

    • Sublime REPL–have REPL right in your sublime!

    • SublimeGutter - view changes in git instantly in your editor

    • CanISwitchToSublimeText3 - This will help me decide.

    • MacTerminal – open this folder in terminal. (Support iTerm2)

    • Local History

    • VAlign easily vertically-align the text around the cursor with cmd+\

    • TidyTabs – A simple keystroke ctrl + alt + w will close all tabs whose file has not been modified in the last 30 minutes.

    • AutoFileName

  • Version Control

    • SideBarGit – add git support to context menu

  • HTML/Web

    • Tag
    • Emmet – quick html coding
    • browser-refresh sublime – insanely great for web editing
    • livecss - highlights all your hex and rgb colors in css/scss files with that actual color!
    • Hayaku - Fuzzy abbreviations for CSS – make you write stylesheet superfast.

  • Design * ColorPick – using Mac Color Picker in Sublime. (Other Platforms?–try ColorPicker)

  • Language-specific

    • MarkdownEditing

    • Python: SublimeRope

    • Syntax Highlighters: JQuery, Less, CoffeeScript, HTML5, Syntax Highlighting for Sass

    • Snippets: HTML5, Bootstrap,
